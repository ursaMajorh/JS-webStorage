Web Storage - JavaScript

Web Storage merupakan sebuah mekanisme di JavaScript yang memungkinakan untuk menyimpan pasangan key-value. key value ini mewakili objek penyimpanan yang selalu berupa String

localStorage vs sessionStorage

  • sessionStorage :

    hanya menyimpan saat browser terbuka, jika meng-close hrowser, lalu masuk lagi ke browsernya, maka secara otomatis penyimpanan yang di simpan di sessionStorage akan terhapus

  • localStorage :

    data akan terus tersimpan di browser walupun browser-nya sudah di close, dan tidak ada batas waktu selama tidak menghapus window-nya
